6.2.1 feature scaling¶

  • 서로 다른 변수의 값 범위를 일정한 수준으로 맞추는 작업

  • 표준화(standardization), 정규화(normalization)

normalize <- function(x){
  return ((x - min(x)) / (max(x) - min(x)))
}
 
standard <- function(x){
  return ((x-mean(x)))/(sd(x))
}

iris[,1:4] <- lapply(iris[,1:4],normalize)
